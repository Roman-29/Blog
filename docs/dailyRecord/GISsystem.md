# GIS 体系介绍

## 地图数据生产

### 数据获取

从获取方式来看可以分为:

1. 矢量化：纸质地图扫描后，进行配准投影及数字化处理 
2. 测量：导出全站仪等测量仪器采集的点数据，进行内业成图处理 
3. GPS采集：利用GPS接收机或RTK技术采集的数据，进行内业成图处
4. 遥感影像解译：利用卫星影像，数据预处理，进行影像自动分类识别或人工解译提取信息 
5. 航测或雷达数据：利用航拍的卫片或利用雷达数据，提取信息

从获取途径来看可以分为：

1. 测绘部门
2. 商业机构
3. 独立采集
